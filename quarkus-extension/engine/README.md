# OrqueIO Engine Quarkus Extension

You can add the OrqueIO Engine Quarkus Extension into your Quarkus application by adding
the following dependency into your `pom.xml` file.

```xml
<dependency>
  <dependency>
    <groupId>io.orqueio.bpm.quarkus</groupId>
    <artifactId>orqueio-bpm-quarkus-engine</artifactId>
    <version>${version.orqueio}</version><!-- place orqueio version here -->
  </dependency>
</dependency>
```
